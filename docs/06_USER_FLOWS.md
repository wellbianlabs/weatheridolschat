## 3. 화면 상세

### S00 · Splash
- 로고 + 그라데이션. 0.5~1s 내 라우팅.
- 분기: `session && profile.onboarded_at` → S04, 세션 없음 → S02, 세션 있고 미온보딩 → S03.

### S01 · 랜딩 (웹 전용)
- 히어로: "오늘 날씨, 누구랑 이야기할래?"
- 4인 캐릭터 카루셀, "지금 만나러 가기" CTA → /login.
- SEO를 위해 SSG.

### S02 · 로그인
- Google 로그인 (Primary)
- 이메일 + 비밀번호 (Secondary)
- 약관·개인정보 동의 인라인 체크
- 익명 체험(Phase 2): 24h 게스트.

### S03 · 온보딩 (3 step 풀스크린)
1. **Step 1 — 닉네임 & 성별/생일**
   - 닉네임 중복 확인
   - 호칭(오빠/언니/이름) 선택
2. **Step 2 — 위치 허용**
   - GeoLocation 권한 요청
   - 거부 시 도시 검색으로 폴백
3. **Step 3 — 마음에 드는 캐릭터 선택**
   - 4인 큰 카드, 사진 + 한 줄 소개
   - 선택 시 첫 채팅 세션 자동 생성 → S05로 이동

### S04 · 캐릭터 선택 (홈)
- 상단: 날씨 배지(현재 위치 · 온도 · 컨디션) — 탭 시 S10
- 중앙: 4인 그리드 (2x2). 각 카드 = 캐릭터 일러스트 + 오늘의 짧은 인사 1줄 (마지막 응답 미리보기) + 미읽음 뱃지.
- 하단: "오늘의 픽" — AI가 날씨 기반으로 추천한 1명에 펄스 애니메이션.
- 우측 상단: 설정 아이콘 → S09.

### S05 · 채팅
**레이아웃:**
```
┌────────────────────────────────────────┐
│  [←]   써니           [⋯]              │ ← 헤더 (캐릭터·메뉴)
│  ☀️ 부산 25°C · 맑음                    │ ← 날씨 컨텍스트 바
├────────────────────────────────────────┤
│                                        │
│   ◯ 안녕! 오늘 햇살 진짜 좋아 ✨        │ ← 어시스턴트 버블
│                                        │
│         반가워! 오늘 뭐해? │← 사용자 버블│
│                                        │
│   ◯ [이미지 카드 1024x1024]            │
│                                        │
│   ◯ [상품 카드: 코랄 립밤]              │
│                                        │
├────────────────────────────────────────┤
│  [📷] [🎁] [🎵]  메시지 입력...   [↑]  │ ← 멀티모달 숏컷 + 입력
└────────────────────────────────────────┘
```

**다이내믹 배경:** 현재 날씨에 따라 그라데이션 + 파티클 (비 → 빗방울, 맑음 → 햇살, 흐림 → 안개, 천둥 → 번개 플래시). ASMR 옵션 ON 시 배경 사운드 루프.

**상호작용:**
- 입력창 좌측 아이콘:
  - 📷 = "셀카 보여줘" 단축 발화
  - 🎁 = "오늘 추천 뭐 있어?"
  - 🎵 = Phase 2 (Suno)
- 메시지 길게 누르기: 복사 · 신고 · 재생성(직전 어시스턴트만).
- 스크롤 끝까지 위로 → 이전 페이지 fetch.

**스트리밍 UX:**
- 메시지 보내자마자 사용자 버블 즉시 표시 (낙관적 UI).
- 어시스턴트 버블 자리에 typing dots → 첫 토큰 도착 시 텍스트 점진 렌더.
- 첨부(이미지/상품)는 done 이벤트 후 슬라이드 인.

**에러 상태:**
- 네트워크 끊김 → 토스트 + 재시도 버튼.
- 429 rate_limited → 페이월 시트 S07 자동 오픈 (free일 때).
- 451 safety_blocked → 페르소나 거절 응답이 그대로 표시 (사용자에게는 정상 메시지로 보임).

### S06 · 이미지 모달
- 풀스크린 이미지, 핀치 줌, 다운로드, 공유.
- "이 사진 한 장 더" → 동일 프롬프트로 재생성(쿼터 차감).

### S07 · 페이월 시트
- 모달 하단 시트(웹은 다이얼로그).
- 헤드: "써니랑 더 깊게 이야기하려면 ✨"
- 비교 표 (Free vs Premium):
  - 메시지 한도 30/일 → 무제한
  - 모델: 빠른 Gemini → 깊이 있는 Claude
  - 이미지 3/일 → 50/일
  - 장기 기억 ✕ → ✓
  - 광고 표시 → 제거
  - 화상통화·노래·영상 (Phase 2) coming soon
- CTA: "월 9,900원으로 시작" → S11 (Phase 2). MVP는 "관심 등록" 버튼 + Waitlist 저장.

### S08 · 퀘스트
- 카드 리스트:
  - "오늘 날씨 제보하기 (+30 토큰)"
  - "캐릭터에게 인사 3회 (+10)"
  - "친구 초대 (+100)"
- 진행도 바 + 보상 수령 버튼.

### S09 · 설정
- 프로필 (닉네임·생일·언어·시간대)
- 위치 (S10)
- 알림 (모닝콜·날씨 알림·푸시 허용)
- 안전 (성인 콘텐츠 차단 등)
- 구독 관리 (Phase 2)
- 토큰 지갑 (Wellbian 잔액)
- 약관/정책 (S12)
- 로그아웃 / 탈퇴

### S10 · 위치 선택
- 현재 위치 사용 (GPS)
- 도시 검색 (Mapbox 또는 카카오 로컬 API)
- 즐겨찾기 ("집", "회사") 다중 등록 → 채팅에서 위치 컨텍스트 토글.

### S11 · 결제 (Phase 2)
- 플랜 선택(월/연)
- 결제 수단 (토스/Stripe/IAP)
- 약관 동의 → 결제 → 완료 화면.

### S12 · 법적 고지
- 이용약관, 개인정보처리방침, AI 응답 안내, 미성년자 정책, 환불 정책.

## 4. 모바일 네비게이션

하단 탭(3개):
- **홈** (S04)
- **퀘스트** (S08)
- **설정** (S09)

## 5. 웹 네비게이션

- 좌측 사이드바(데스크톱) 또는 햄버거(모바일 웹):
  - 홈 / 퀘스트 / 설정 / (로그아웃)
- 채팅 화면은 좌측 사이드바 유지, 우측이 메시지 패널.

## 6. 빈/에러/로딩 상태 컴포넌트 매트릭스

| 화면 | Empty | Loading | Error |
| :-- | :-- | :-- | :-- |

| 채팅 메시지 | "써니에게 첫인사 해볼까?" + 추천 prompt 3개 | 스켈레톤 3 버블 | 토스트 + 재시도 |
| 퀘스트 | "아직 진행 중 퀘스트가 없어요" | 스켈레톤 | 동일 |
| 이미지 모달 | — | 블러 + 진행률 | "이미지 생성 실패 — 다시" |

## 7. 알림 & 푸시 (옵트인)

- **모닝콜(07:30)**: 캐릭터별 한 줄 + 오늘 날씨 요약 → 탭 시 S05.
- **날씨 급변**: 현 위치에 강수 시작/태풍 시 알림 (1시간 1회 제한).
- **퀘스트**: 미수령 보상 18:00 리마인드.
- **결제 알림**: 갱신 7일 전.

iOS APNs / Android FCM / Web Push (Notification API). Expo Push Service로 추상화.

## 8. 접근성

- 모든 인터랙티브 요소 최소 44x44dp.
- 명도 대비 WCAG AA.
- 채팅 메시지 ARIA `role="log"` + `aria-live="polite"`.
- 다이내믹 배경 모션은 prefers-reduced-motion 존중.
