# 00 · Product Overview

## 1. 비전 (Vision)
> **"날씨로 시작해 일상으로 이어지는, 4명의 아이돌과 함께하는 초개인화 라이프 컴패니언."**

K-weather의 초국지적(hyperlocal) 기상 데이터와 멀티모달 AI(텍스트·이미지·영상·음악)를 결합하여, 사용자의 감정·일정·소비를 자연스럽게 케어한다.

## 2. 핵심 가치 제안 (Value Proposition)

| 페인 포인트 | 우리의 해결 |
| :-- | :-- |
| AI 챗봇과의 대화가 어색하고 시작이 막막함 | **날씨**라는 보편적 소재로 자연스러운 ice-breaking |
| 일반 챗봇은 페르소나가 얕고 기억이 없음 | **4인 아이돌 그룹** 페르소나 + 프리미엄 장기 기억 |
| 추천이 광고 같고 이질적 | 캐릭터가 **'선물' 카드** 형태로 자연스럽게 큐레이션 |
| 단순 대화로는 몰입이 약함 | **이미지·영상·음악** 즉시 생성으로 멀티모달 체험 |

## 3. 타깃 (Target Persona)

- **1차:** 10대 후반 ~ 20대 후반 K-pop·서브컬처 친화 사용자 (한국 중심, 일본·동남아 확장)
- **2차:** 30대 이상 감성적 라이프 컴패니언 수요자 (심리 상담·힐링 콘텐츠)

## 4. 캐릭터 그룹: Prism Station

총 4명, 각 멤버는 날씨 모티프 + 출신 지역 + 성격 + 추천 도메인이 묶인 1:1 매핑.

| 멤버 | 모티프 | 출신 | 핵심 무드 | 주력 추천 도메인 |
| :-- | :-- | :-- | :-- | :-- |
| **써니 (Sunny)** | ☀️ 햇살 | 부산 해운대 | 긍정·활동적 리더 | 아웃도어/뷰티/운동 |
| **레인 (Rain)** | 🌧️ 비 | 일본 가나자와 | 차분·시적·공감 | 홈카페/힐링/심리 |
| **클라우디 (Cloudy)** | ☁️ 구름 | 춘천 | 몽환·엉뚱·예술 | 전시/감성 핫플 |
| **썬더 (Thunder)** | ⚡ 천둥 | 상파울루·이태원 | 카리스마·반항 | 배달/게임/스트레스 해소 |

전체 페르소나 디테일은 [05_AI_PROMPTS.md](05_AI_PROMPTS.md) 참고.

## 5. 사용 시나리오 (Day-in-the-Life)

1. **08:00 모닝콜** — 써니가 "오늘 부산 25도, 미세먼지 좋음! 러닝 어때?"
2. **12:30 점심** — 레인이 "비 와서 따뜻한 우동 어때? 근처 ○○집 평점 4.6" (Nasmedia 추천)
3. **19:00 퇴근길** — 클라우디가 오늘의 노을 사진(이미지 생성) + 그림 같은 카페 추천
4. **22:00 위로** — 썬더가 스트레스 해소 EDM 플레이리스트(Suno) + 짧은 안무 영상(SeaDance)

## 6. 핵심 지표 (North Star Metrics)

| 단계 | 핵심 KPI | MVP 목표 |
| :-- | :-- | :-- |
| Acquisition | 신규 가입 / 주 | — |
| Activation | 첫 7일 내 3회 이상 대화 | 40% |
| Retention | D30 리텐션 | 25% |
| Revenue | ARPPU (프리미엄) | ₩9,900/월 |
| 멀티모달 사용 | 사용자당 주 1회 이미지·음악 생성 | 30% |

## 7. 차별화 포인트

1. **실시간 기상 컨텍스트**: 단순 LLM이 아닌 K-weather API 기반 진짜 현재 날씨로 응답.
2. **시각적 캐릭터 일관성**: OpenAI Reference Image 기반 시드 고정으로 매번 같은 얼굴.
3. **네이티브 커머스**: 광고가 아닌 캐릭터의 "선물"로 자연스럽게 노출 (Nasmedia 연동).
4. **Web3 리워드**: 기상 제보 퀘스트 → Wellbian 토큰 → 광고 수익 환원.

## 8. 범위 명시 (In/Out Scope)

### ✅ MVP 범위
- 캐릭터 4인 선택 / 1:1 텍스트 채팅
- 날씨 API 연동 → 캐릭터 응답에 자동 주입
- 이미지 생성 (캐릭터 셀카) — Mock으로 시작
- Freemium 진입 (게이팅 UI만)
- 구글 OAuth + 이메일 로그인

### ❌ MVP 제외 (Phase 2+)
- 실시간 화상통화 (OmniHuman)
- 안무 영상 생성 (SeaDance)
- 음악 생성 (Suno)
- Web3 토큰 실제 발행 (시뮬레이션만)
- Stripe/토스 실결제

## 9. 글로벌화 / i18n

- **언어:** ko, en, ja (Phase 2)
- **타임존:** 사용자 디바이스 기준
- **날씨 API:** 한국은 기상청+K-weather, 해외는 OpenWeatherMap fallback

## 10. 위험 요소 (Risk Register)

| 위험 | 영향 | 완화 |
| :-- | :-- | :-- |
| LLM 환각 / 부적절 응답 | 브랜드 손상 | 3중 필터링 + Graceful Refusal ([09_SAFEGUARDS.md](09_SAFEGUARDS.md)) |
| AI API 비용 폭증 | 마진 악화 | 무료 티어 Gemini 라우팅, 토큰 길이 제한 |
| 캐릭터 이미지 불일치 | 몰입 저하 | Reference Image + seed 고정 |
| 광고 거부감 | 이탈 | 네이티브 커머스 디자인, 빈도 제한 |
